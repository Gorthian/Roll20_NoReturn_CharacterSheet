Würfelhintergrund

Problem

Eine Folge von Zahlen soll über CSS so angepasst werden, dass jede einzelne Ziffer als Würfel dargestellt wird.
Hinter der Ziffer soll also z.B. das Piktogramm eines Würfels erscheinen, oder die Zahl zumindest von einem quadratischen Rahmen begrenzt sein.
Dabei ist nicht bekannt aus wievielen Ziffern die Zeichenfolge besteht.
Die Ziffernfolge und der umschließende SPAN-Tag können auch nicht bearbeitet werden.
Javascript steht nicht zur Verfügung.
